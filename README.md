## Front-end Tech Stack

- JavaScript(ES6)
- HTML5
- CSS3
- Sass
- React.js
- Redux
- React Hooks
- Local storage
- Web API
- Axios

## Features
- Fetch data from **blockchain APIs**.
- **Graphs** that displays the market values - used **Recharts library** to display data.
- **Contact CRUDL** (create, read, update, delete, list).
- **Filter** contacts
